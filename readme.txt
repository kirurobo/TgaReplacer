■ 概要
DLLなどバイナリファイル中に含まれる 1024×512px 32bit TGA画像らしき領域を別の画像で置き換えるツールです。

■注意
本ツールはバイナリファイルを書き換えるため予期しない結果をもたらす可能性があります。
対象のファイルはバックアップを必ずとってください。
本ツールの利用で生じる影響について作者は一切責任は負えません。


■ 使い方
1つ目のテキストボックスに、OculusPlugin.dll をドロップします。
2つ目のテキストボックスでは、表示させたい画像を選択しておきます。
変換開始を押して、上手く行けば保存するか聞かれます。
保存すると元のファイルは置き換えられます。

■ その他説明
SDKをみると同名で異なる OculusPlugin.dll があるため、
作品を公開用にビルドした中に含まれる OculusPlugin.dll を書き換えると良い気がします。

SDK 0.4.2 で試験しています。
それより前のバージョンで OculusPlugi.dll が8.5MB程度あるものは、
中の画像が非圧縮で、用意する画像も非圧縮にしないと上手く行きませんでした。
公開されている Dogelus2 や Mikulus がそうなっているようです。
